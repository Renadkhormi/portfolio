# Renad Khormi Portfolio

This portfolio was developed as part of the Web Development course (CCSW321). It presents my work, skills, and interests in software engineering with a focus on UX and UI design.

## Live Website
https://renadkhormi.github.io/portfolio/

## About the Project
This website reflects my interest in designing user-centered digital experiences. The goal was to create a clean and structured interface that presents information clearly while maintaining a simple and modern design.

## Features
- Responsive design that works across different screen sizes
- Clear and organized layout
- Focus on usability and readability
- Simple navigation between sections
- Use of Bootstrap components to enhance layout and structure

## Technologies Used
- HTML5
- CSS3
- Bootstrap

## Sections Included
- About Me
- Education
- Projects
- Certificates
- Activities
- Contact

## How to Run the Project
1. Download the project files
2. Open the `index.html` file in your browser
